# MatScan
A multi-threaded rust material scanner

Usage:
Put the script.json from the Il2cpp dump
Open cmd Go to the folder where the scanner belongs and write: Scanner.exe script.json 1000 3 4

1000: class scan range
3: class scan depth
4: thread count
